Estudo de Caso: Proposta Inicial

Tema: Loteria Online (Virtual)

Resumo do Projeto

    O sistema que está proposto para estudo de caso neste semestre letivo é contextualizado, de forma mais ampla, como um sub-sistema componente do Sistema Brasileiro de Loterias, cujo órgão regulador é a Caixa Econômica Federal (CEF). Trata-se de um sistema de coleta de apostas, através de uma loja (lotérica) virtual e registro instantâneo da mesma nos sistemas de loterias da CEF. 

    As regras definem que cliente da lotérica virtual deve manter uma conta com a loteria virtual (nos mesmos moldes das contas do sistema de telefonia celular pré-paga adotado no Brasil). Esta conta permite a aquisição de créditos, mediada pelo Sistema de Pagamento Brasileiro, e débitos correspondentes a cada aposta realizada pelo apostador.

    Algumas funcionalidade extras são oferecidas, tais como consultas a resultados e concursos anteriores e regaste de prêmios de menor valor.
